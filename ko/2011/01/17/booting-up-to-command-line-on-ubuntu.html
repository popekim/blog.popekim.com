ubuntu 10.10 커맨드 라인으로 곧바로 부팅하는 법

김포프 2011-01-17
  • linux

주말동안 드디어 우분투 10.10을 깔았다. 태어나서 처음으로 리눅스를 깔게 된 주 이유는
웹프로그래밍 테스트 서버 용도였기 때문.

서버로 쓸 거라 GUI가 필요 없어서 Server Edition을 깔고,
그 위에 데스크탑을 별도로 설치했는데…

제길 --; 시작할 때마다 GUI 로그인 화면이 뜬다. 써글… --

GDM을 끄려고 구글링을 했으나
모두 이전 버전용 방법이거나, 서버말고 처음부터 데스크탑 설치 시만 가능한 방법뿐이었다.

결국 리눅스/유닉스 대가 직장동료 데니엘 박사님의 도움을 받아 해결.
같은 문제 겪는 분들 위해 공유한다.

Ubuntu 10.10에서 GUI(gdm) 안 뜨게 부팅하는 법

1) gdm.conf 수정

sudo vi /etc/init/gdm.conf

2) 아래 라인을 찾는다

start on ( filesystem

다음처럼 수정:

start on (runlevel [5]
          filesystem

3) 아래 라인을 찾는다

stop on runlevel [016]

다음처럼 수정:

stop on runlevel [01236]

이렇게 하면 모든 것이 유닉스와 동일한 run level 부팅 방식으로 동작한다.

구글 검색에서 stop on runlevel [0126]로만 바꾸라고 해서
그렇게 했다가 부팅 시 컴퓨터가 멈췄다 -_-;;

이 방식은 내 PC에서 잘 동작한 설정.
다른 분들에게도 도움이 되길!
