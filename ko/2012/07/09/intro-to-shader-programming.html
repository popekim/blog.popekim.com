셰이더 프로그래밍 입문 (도서평 4.8 / 5.0)

한빛미디어에서 제 책을 출판해주었습니다. 제 블로그에 비해 바뀐 내용도 좀 되고, 편집도 상당히 괜찮게 나왔으니 꼭 구매해 보시길 바랍니다. 불필요한 내용도 빼서 가격도 꽤 저렴합니다. ^^ (저도 먹고 살아야…… -_-)

책에 대한 자세한 소개는 YES24 페이지에 있는 '지은이의 서문'을 참고해주세요.

  • 도서구매 (YES 24)
  • 소스코드 샘플: Zip, GitHub
  • 1쇄/2쇄 정오표
  • 도서평 모음 평점 4.8 / 5.0 (2012년 11월 현재)
  • 비공식 GLSL 소스코드 샘플 (독자분 중 한분이 직접 만드셨습니다)
  • 렌더몽키 다운받기
